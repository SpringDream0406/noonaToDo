# 기능
- 유저는 할일을 추가할 수 있다
- 각 할일에 완료와 삭제 버튼이 있다
- 완료버튼을 누르면 할일이 끝난것으로 간주하고 밑줄이간다
- 완료된 할일은 되돌리기 버튼을 클릭하면 다시 되돌릴 수 있다
- 탭을 이용해 아이템들을 상태별로 나누어서 볼 수 있다
- 할일을 추가할 때 배경의 별똥별이 할일탭으로 날아오는 git로 변경된다
- 삭제 버튼을 누르면 진짜 삭제할건지 물어본다
